# Day 10: Data Chatbot (CSV Based)

## 📌 Description
This project is a simple data-driven chatbot that reads a CSV file and answers user queries based on the data.

## 🚀 Features
- Reads student data from CSV file
- Calculates average marks
- Finds highest marks
- Displays student names

## 🛠️ Technologies Used
- Python
- CSV module

## 📂 Files
- day10.py → Main chatbot code
- students.csv → Dataset file

## ▶️ How to Run
1. Open terminal in this folder
2. Run:
   python3 day10.py

## 💡 Example Inputs
- average marks
- highest marks
- names

## 📈 Output Example
You: average marks  
AI: Average marks = 78.75
